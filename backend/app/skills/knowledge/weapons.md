# Foil (花剑) Knowledge

## Target Area
- Only the torso (torso is valid target)
- Head, arms, and legs are not valid

## Key Techniques

### Footwork
1. **En Garde** - Basic starting position
2. **Advance** - Moving forward
3. **Retreat** - Moving backward
4. **Lunge** - Primary attacking technique
5. **Recovery** - Returning to en garde

### Blade Work
1. **Straight Cut** - Direct attack
2. **Cutover** - Blade change attack
3. **One-Two** - Beat followed by attack
4. **Remise** - Continued attack after missed
5. **Counter-Riposte** - Return attack after parry

## Priority Rules
- Attack has priority over counter-attack
- Compound attacks lose priority when beat occurs
- Parry gains right of way

## Common Mistakes
1. Over-extending on lunge
2. Crossing feet during advance
3. Flat-footed stance
4. Late recovery

# Épée (重剑) Knowledge

## Target Area
- Entire body is valid target
- No priority rules - first touch scores

## Key Techniques

### Footwork
- Same as foil but with more defensive positioning
- Emphasis on distance control

### Blade Work
1. **Counter-Attack** - High value in épée
2. **Time Attack** - Taking advantage of opponent's setup
3. **Distance Lunge** - Longer lunges common

## Strategy
- More tactical than other weapons
- Defensive play more viable
- Counter-attacking is key

# Sabre (佩剑) Knowledge

## Target Area
- Upper body including head
- Cuts and thrusts both valid
- Head valid target

## Key Techniques

### Footwork
- Fast, explosive movements
- Quick direction changes
- cavalry attacks and retreats

### Blade Work
1. **Cut** - Primary attack method
2. ** Parry** - Blocking with guard
3. **Riposte** - Return after parry

## Special Rules
- 0.2 second right of way after attack lands
- Cuts to the back are not valid
- Cavalry attacks (mounted) not used in modern fencing
